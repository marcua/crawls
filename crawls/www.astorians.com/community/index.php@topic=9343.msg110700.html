First. Server got hacked (again). And the best thing the support department could tell me to do to "fix" the problem was to start from scratch with the whole site. So with that I decided maybe we had outgrown that provider and I should move to a new one.

Selected a new provider and got the data migrated over, then went to update the domain record to point to the new server and lo and behold the domain provider had a hardware failure that prevented (and still is preventing) the records from updating.

Next, this morning, after getting all the bugs worked out of the new server install I got up to see that the entire website and all data had been deleted. When I talked to the new provider about it they claimed their backup had gotten corrupted. So, I ended up having to install from the latest back up I had, which - was from the old server.

However - I'm not really too impressed with the new provider either. Unfortunately the site is big enough that we don't fit on a small and cheap provider, but it's not quite big enough for a full blown dedicated server. Just right in the middle.
