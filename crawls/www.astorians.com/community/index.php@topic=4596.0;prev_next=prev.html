Random calls to cell phone?
« on: January 07, 2007, 08:22:43 PM »
Not sure where to post this - but has anyone ever experienced problems with repeated random calls to your cell phone?  A couple of weeks ago, I received a call from an unfamiliar number.  When I picked up, I heard a hanging-up click.  The number was very long, so I thought it was an international number, and some sleuthing determined that the first three digits were the country code for the Ukraine!

Today, I received several more calls from the same number - over a dozen.  Each time, there was a click or if I didn't pick up, it registered as a missed call.  I called T Mobile to ask whether they could block incoming calls from this number, but they said it wouldn't be possible and that the only solution, if the problem continues, is to change my cell phone number.

I should also add that the first call woke me up in the wee hours of the morning, so that was a real pain and an experience that I hope not to repeat again.  Any ideas, anyone??
