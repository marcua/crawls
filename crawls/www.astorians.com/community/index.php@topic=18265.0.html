Which wireless doorbell do you have? (Company, model, etc) Maybe you just got a bad one if it's been happening all the time.

Although I don't know too much about doorbells, you should really try to get the doorbell wired if you can. Wired doorbells are more reliable than wireless because there won't be interference from the bricks, etc.

If you have to go wireless, this particular doorbell on Amazon seems to have great reviews.
