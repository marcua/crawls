Before the Vault & set into a giant marble pedestal
are touch stones with mysterious characters written upon them
which you must depress in order to discover what is contained within.
Also, once a button is pressed, it remains depressed, so that it cannot be pressed again unless the whole puzzle is reset (by pressing 'P') in which all the doors will return to the closed state. You can reset the keypad as many times as you like.
