Here's one:

You're in a dark room with 50 quarters, 18 of which are heads up. You are allowed to move the coins around or flip some or all of them if you wish. Problem is, it's too dark to tell what you're moving or flipping. Your job is to divide the coins into two groups, but each of the groups must have the same number of heads up coins. How do you accomplish this?

(No, you can't tell which ones are which by "feeling" them. And no, you can't just turn on the light either. This is actually solvable using logic/math.)

Yes. The two groups will equal all 50 coins.

You have to split the coins into two groups, but so that each group has the same number of coins that are heads up.

For example you could split the coins into two groups: one group with 49 coins and the other with 1. That isn't necessarily the answer though.

But the problem is how do you make sure that each group has the same number of heads up coins without being able to see the coins?

yes, but then you'd never know when you had solved the puzzle. You can flip them as much as you want, but at some point you must be able to say for sure that you have completed the task and have an equal number of heads in each group.
