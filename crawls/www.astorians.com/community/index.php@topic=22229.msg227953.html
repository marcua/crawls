So I wanted to get this out there, in case anyone has some names / references that we can start pestering, but since we have some big things happening in the next year (a wedding being the biggest) I figured it wouldn't hurt to try and get a list of realtor / landlord / whatever names together as we're looking to move by March of 2013.
