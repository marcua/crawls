when you set up the game for the first card you place you have 52 cards to choose from... for the next card you have 51 cards... and then 50 and so on until you place all the cards (including those that are not set up on the "board" but you're holding them in your hand.  Still each arrangement in your hand counts as a different set up).

If the game involved only 2 cards you'd  have 52 x 51 different ways of setting up
3 cards would mean 52 x 51 x 50
4 cards 52 x 51 x 50 x 49
etc...
because you are using all the cards in the deck it's 52 x 51 x 50 x ... 3 x 2 x 1  or 52! (52 factorial - not 52 YOWEEE!!!!)

52! is approximately 8 x 10^ 67 .... which is a hellava lot more than 1000000 but a lot less than 1.7  x 10 ^89 (52 to the 52nd power)
