Post by: merm on September 08, 2006, 07:21:21 AM
Hi guys,

Yeah, if you click on that "check astorians" link through dnsstuff.com it will say the same thing for everyone. Doesn't actually matter which service provider you are reaching us through, I just asked that early on to try and pinpoint the problem in case it was with people's local ISPs - it's not.

What's happening now (the reason some of you can access and some can't) is that despite the fact that the address has been removed from the root DNS servers, some of your computers and ISPs are "remembering" the address from when before it was deleted. Every computer "caches" the address for a while, and most ISP's do as well. Commercial ones cache DNS for longer, which is why some of you can access from work. At this point though DNSStuff is going to report the same for all of us because it's not reporting on your local internet connection, but that is the report for the ROOT domain servers for the entire internet. If they can't see us (they say "astorians.com doesn't exist") then nobody can.

No word yet on a cause, so I don't want to start any rumors.
