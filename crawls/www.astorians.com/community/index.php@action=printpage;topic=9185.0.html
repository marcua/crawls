Title: Re: 30th Ave NYSC - How busy?

At that time, I'd say the biggest congestion is at the treadmills. It can be hard sometimes to find an open one (that works, of course). I've only used the treadmill there once or twice, though, so I can't really say how long a standard wait would be. The weight machines seem relatively easy to use, and I've never had a problem with the showers. There can be congestion in spots but nothing like midtown.
