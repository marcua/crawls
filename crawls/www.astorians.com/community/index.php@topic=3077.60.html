I really appreciate all the input on this.

I'm not completely against eventually looking at the option of advertsing for the site, but if we can avoid it I think it would be cool.

For now I think I'm just going to add a "donate" button to the site so that anyone who is inclined to support the site can do so. But I'd like to keep it as simple and unobtrusive as possible.

We're not talking about substantial equipment costs to operate the site. It's somewhere around $600/year. There is a significant time investment for wasabisam and I, but it's something I enjoy doing (though I'd love to do it for a living, but that's just a dream). The other mods also really contribute a lot in keeping things organized and I'm really thankful for that.

Donate button will appear in the next few days.

In other news, we are definitely moving to another server. That should happen in another day or two. The current host I'm with (Site5) keeps disabling the site because we're using too many server resources. I'm not too happy with their plan at the moment. But I've got a brand new server all ready for us that should speed things up a bit. And we've got guaranteed memory and CPU priority that will mean that someone can't just turn the site off when they think we have too many users online. I'll provide another update right when I migrate the site to the new server because you'll notice some anomalies while It's in progress.
