Self-defense class for boys and teens teaches violence avoidance and detection, equipping boys with the tools to prevent violent confrontations and how to deal with the threat of physical violence.

Curriculae: Violence avoidance, using talking to de-escalate a situation, techniques to deal with physical confrontation (shock and stop). The techniques will be practical street defense moves (based on krav maga).
 
Time investment: Total 8 classes - Two 50 mins classes a week (weds 5.30p & fri 6.30p), for 4 weeks
Student must be able to attend both classes a week, it is not an either-or option.

Title: Re: Free Self-Defense Classes for Boys, ages 8-14
Post by: Dolly on November 23, 2010, 03:24:16 PM
It's an all-male class with a male instructor.
1) Some boys feel uncomfortable pairing up with girls for pratice of self-defense techniques, since it involves physical contact, however minimal.
2) An all-male class has a different dynamic from a mixed class and allows boys to ask a greater variety of questions unhindered by societal norms.
3) Some families have religious views and prefer a male only class.

We have self-defense classes for different groups at different times. We have had a women's self-defense course taught by a female instructor. In the future there will be self defense classes for girls only (8-14) - for the same reasons as above, a mixed class for youngsters (4-7), etc. Dates undetermined.
