Title: Astorians 3.0 - Site Upgrade
Post by: merm on August 22, 2006, 09:50:49 AM
Greetings,

We're proud to announce that a major update to Astorians is in the works! We're calling this Astorians 3.0. The new look we've been working on is now the default theme on Astorians for all users and guests.

There's a few things you should know about Astorians 3.0:

  • Astorians 3.0 is now the official theme for Astorians.
  • If you prefer the Astorians 2.0 interface (the old one), you are welcome to use it (the same goes for those that use the SMF Default Theme). You are under no obligation to utilize the new features and theme. Just simply select "Astorians 2.0" per the instructions below. You should note however that Astorians 2.0 will no longer be upgraded or supported.
  • We've also added a couple new categories
  • Feedback - We welcome your feedback and impressions.

INSTRUCTIONS for changing your theme:

1. Click on the button on the top of the scren that says "PROFILE"
2. On the left menu bar select "Look and Layout Preferences"
3. Where it says "Current Theme" select the highlighted text that says "Change"
4. You will now see a list of available themes. Select the one you like best.

* Special thanks to our very own wasabisam for the great work she's done making both Astorians 2.0, and the new theme as well *
