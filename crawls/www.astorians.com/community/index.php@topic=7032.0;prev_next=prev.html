Being that this is an older computer, the cord that you need to connect the CD-ROM drive to the motherboard (MB is called an IDE cable.  Of course, there are a couple of different types of IDE cables.  Newer IDE cables are easy to find in any COMPusa store, but they may not work with older hard drive's (HD).  It may be more difficult to find an older one.

Sometimes, depending on how the computer is configured, the same cord that connect the floppy drive to the MB also has an extension which reaches the HD.  Look inside the case and see if the floppy drive cord has an extension which reaches the HD, or even a skinny plastic piece with a small strip of holes which can connect to the HD.  If the cord doesn't have this, then you need to get an IDE cable.  They cost around $5 bucks.

As for the power cable; usually, printer power cords (generally speaking) also work with CPU's since they use the same three prong interface.  If you have a printer, try using that cord.  If not, this can also be obtained from COMPusa for around $10.

So the most you might have to spend is $10, which really isn't so bad...even if the computer doesn't work.
