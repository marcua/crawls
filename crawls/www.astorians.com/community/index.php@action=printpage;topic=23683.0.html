I would never trust computer repair people. And with an 8 year old computer, I wouldn't bother spending any more money on it anyways. If you're seeing strange issues like overheating and disappearing hard drive space, it probably means the physical hardware is starting to fail. Once you add up the costs of parts and service, it will easily get close to or go beyond the cost of buying a new computer. And at that point, you'll realize that you spent all this money and you still have an 8 year old computer that might start having other parts fail.


I'm a Mac guy but I understand that they're too expensive for some people. Since I haven't been a student for over 10 years, I personally don't have a need for a laptop, so I've been buying desktops for a while now. Desktops are cheaper to purchase and maintain than laptops. As I mentioned, I'm a Mac guy so I've been using a Mac Mini for a while. You can get one for $600 and just get a generic (non-Apple) keyboard, mouse and monitor for about $150 total. This is the cheapest Mac computer you can buy. A Dell desktop will probably run you about $500 with monitor.


If you have to get a new laptop, Mac laptops start at around $1000 and bare-bones Windows laptops probably start at $500.

Post by: murph on November 07, 2012, 04:09:11 PM
I do everything myself. Don't trust those guys either.

Aside from the CPU issue (good luck with that - not much you can do about laptop CPU hardware issues), it does sound like your PC needs a good clean up or better yet, a clean Windows install. That's what I would really recommend - backup data to external drives or USB sticks, clean install Windows again, restore apps and data. But I understand that's not always possible. So check out the following apps...

CCleaner - good tool to cleanup PC
Malware Bytes - scans for spyware/ad serving software
Autoruns - shows you all apps that load when your PC boots up

Make sure you have current anti-virus and updated definitions. If your AV has expired, uninstall it and download Microsoft Security Essentials (it's free).

As for old updates - if you have Windows XP check c:\windows for a bunch of folders in BLUE. It is safe to delete most of these. If you want to be extra careful - keep the most recent 1-2 months of these, delete the rest.
