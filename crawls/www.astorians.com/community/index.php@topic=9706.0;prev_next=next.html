It's worth keeping in mind that there is no particularly "secure" physical connection to the internet - that's like finding a secure mailbox for sending a post-card: any number of people may still be able to see it along the way.

 A "secure" internet connection is a matter of the computers at each end being secure, and having the data exchange encrypted to prevent eavesdropping.  Tax-filing sites will use "secure http" (urls beginning https:// ) to encrypt the data you exchange, (just like online shopping site checkouts.)

The far greater risk is faulty, badly configured, or malicious software on your computer, which may expose data stored there.  This risk is present whether you're in the process of filing or not.

I personally would be comfortable using an internet cafe wi-fi connection with my own computer, IF
 
 - current patches/service-packs/updaters for system software are applied
 - the computer's software firewall is enabled,
 - any file sharing, etc network services on the machine are disabled or known to be secured, and
 - (particularly if it's a windows machine) an up-to-date anti-virus program is installed and had been used to scan the thing recently.

The same requirements would apply for a machine at home, or on a corporate network.

(Actually, I am more laid-back than that, and I'd settle for less, but you should not.)

You are right to insist on using your own computer.. there is no telling what awful stuff is installed on a public computer.
