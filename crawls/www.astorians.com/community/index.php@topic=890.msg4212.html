Could be any number of things. I'd recommend finding a new computer, that you know works, then plugging the old drive into the new computer. That way you don't have to boot off the old drive, or use the old hardware, in case there are problems with either some segments of the old drive, or any part of the rest of the old computer. The only way you wouldn't be able to get the data off the old drive then is if the drive itself failed catastrophically, which is possible but not as likely as something getting misconfigured or some other failure...
